# AI-Based Human Emotion Detection System

A full-stack application that uses Deep Learning to detect "Happy" vs "Sad" emotions via webcam or image upload.

## 🚀 Tech Stack

- **Frontend:** React 19, Vite, Tailwind CSS, Framer Motion
- **Backend:** Node.js, Express, MongoDB
- **AI Service:** Python, TensorFlow, Flask, DeepFace
- **Charts:** Chart.js

## 📂 Project Structure

- `/frontend`: React application
- `/backend`: Express API
- `/ai-service`: Python AI logic
- `/docs`: Documentation & Diagrams

## 🛠️ Setup Instructions

1. Clone the repo
2. Install frontend dependencies: `cd frontend && npm install`
3. (Steps for backend and AI service coming soon...)

## 📝 Features

- Real-time Webcam Detection
- User Authentication (JWT)
- Emotion History Dashboard
- Admin Analytics
